Javascript

An object computer programmming language commonly used to create interactive effects within web browsers.

Modulo

An operator that finds the remainder after division of one number by another.

Operators

A symbol that represents a specific action.

NaN

Not a Number, although it is a 'type' of number according to Javascript

Parameter

a variable in a method definition

+=

Shorthand operator for plus one

Function

a named section of a program that performs a specific task.

Method

a procedure that is executed when an object receives a message

Escape

A code, denoted by \, that signals execution of the text command rather than print or display the text characters.

Commenting Code

the practice of adding a text note to source code in order to provide explanatory information, usually about the function of the code.

Chaining Methods

A technique that can be used to simplify code in scenarios that involve calling multiple functions on the same object consecutively.

Argument to a Method

When a method is called, the arguments are the data you pass into the method's parameters

Alert

a method that displays an alert box with a specified message and an OK button.

String

A string is a data type used in programming that represents text rather than numbers.

Booleans

a data type with two values usually denoting true or false

Undefined

a condition where an expression does not have a correct value, although it is syntactically correct.

Variable Naming Conventions

a set of rules for choosing the character sequence to be used for identifiers which denote variables, types, functions, and other entities

Return

a statement that terminates the execution of a function and returns control and/or a value to the calling function
